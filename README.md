Speedrun Stopwatch
A clean, dark-themed stopwatch application designed specifically for speedrunners. Track your runs, splits, and personal bests with keyboard shortcuts for seamless fullscreen gaming.
Features

Precision Timing: Millisecond accuracy (00:00.000 format)
Run Management: Name and describe each run attempt
Split/Lap Tracking: Track individual segments with cumulative and lap times
Personal Best Tracking: Automatically highlights your best time with comparisons
Run History: Complete history of all your attempts with detailed split information
Keyboard Shortcuts: Control the timer without leaving your game
Data Persistence: All runs are saved locally and persist between sessions
CSV Export: Export your run history for analysis or sharing
Dark Theme: Easy on the eyes with earthy tones (browns, beige, greys)

Keyboard Shortcuts

1 - Start Run
4 - Add Lap/Split
9 - Finish Run
0 - Reset

Note: Shortcuts won't trigger when typing in input fields
How to Use

Download the speedrun-stopwatch.html file
Open it in any modern web browser (Chrome, Firefox, Edge, Safari)
Enter a run name and optional description
Press 1 or click "Start Run" to begin timing
Press 4 during your run to add splits/laps
Press 9 when finished to save the run
Press 0 to reset and start a new attempt

Data Storage
All run data is stored locally in your browser using localStorage. Your data:

Persists between browser sessions
Stays on your computer only
Can be exported to CSV at any time
Will be cleared if you clear browser data

Browser Compatibility
Works in all modern browsers:

Chrome/Edge (Recommended)
Firefox
Safari
Opera

Tips for Speedrunning

Keep the app open in a second monitor or window
Use keyboard shortcuts for hands-free operation
Add splits at consistent checkpoints for better analysis
Review your split times to identify improvement areas
Export runs periodically as backup

Technical Details

No dependencies: No installation or internet required

Responsive: Works on desktop and tablet screens
